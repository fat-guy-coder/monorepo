# Blitz — 2D 像素风动作武侠游戏

> 核心体验：**刀光剑影，一击一命。** 在 2D 横版世界中体验武侠的快、狠、准。
> 美术方向：像素风 + 法线贴图 + 动态 2D 光照 → Dead Cells 级别视觉效果

## 一句话

2D 横版动作游戏。扮演武侠角色，用轻击、重击、闪避、格挡的组合击败敌人。像素美术 + 骨骼动画 + 法线贴图光照 = 好看又好玩的武侠世界。

## 为什么是 2D 而不是 2.5D？

2.5D 的真正价值在于**多方向射击**（4-8 向），对武术动作游戏来说：
- 武术的横向攻防（格挡方向、招架、破招、拆招）在纯 2D 更清晰
- 资产量：1 个人物 × N 个动作 vs 2.5D 要 × 4-8 个方向
- 参考：Katana Zero / Mark of the Ninja / Guacamelee 全是用 2D 做武术/忍者动作
- **武术的丰富度取决于动画资源和状态机设计，不取决于维度**

## 核心机制

### 战斗系统（三重攻击体系）

| 攻击 | 按键 | 伤害 | 范围 | 硬直 | 说明 |
|------|------|------|------|------|------|
| 轻击 (Light) | J | 1 | 短 | 小 | 快，可连击，第4击为击飞 |
| 重击 (Heavy) | K | 2 | 中 | 大 | 慢但高伤害，可蓄力 |
| 特殊技 (Special) | L | 3 | 特殊 | — | 消耗气力，方向+特殊键出不同招 |

### 连击系统

```
轻 → 轻 → 轻 → 重      = 基础四连（最后一下击飞）
轻 → 轻 → 重            = 破防三连（第三下破格挡）
轻 → 重                 = 挑空二连（第二下浮空）
↑ + 轻                 = 上挑（对空）
↓ + 轻（空中）         = 下劈（落地范围攻击）
重（蓄力）              = 蓄力重击（破霸体）
Dash + 轻               = 突刺（冲刺攻击）
```

### 防御系统

| 操作 | 效果 | 说明 |
|------|------|------|
| 按住 Shift | 格挡 | 消耗架势槽，免伤但无法移动 |
| 被击中瞬间按 Shift | 精准格挡 (Parry) | 不消耗架势槽 + 敌人硬直 + 反击窗口 |
| Dash | 闪避 | 无敌帧穿过敌人/弹幕 |
| 空中 Dash | 空中闪避 | 空中版闪避，一次跳跃限一次 |

### 资源系统

| 资源 | 上限 | 获取方式 | 用途 |
|------|------|----------|------|
| 生命值 (HP) | 3-5 | — | 归零死亡 |
| 架势槽 (Posture) | 100 | 随时间恢复 | 格挡消耗，归零 = 破防 |
| 气力 (Qi) | 3 | 攻击命中恢复 | 特殊技消耗 |

### 角色移动

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | A/D 或 ←/→ | 水平移动，有加速度和惯性 |
| 跳跃 | Space/W/↑ | 可变跳跃高度 + coyote time + jump buffer |
| 冲刺 | Shift（短按） | 无敌帧横向爆发，有冷却 |
| 蹬墙跳 | 贴墙时按跳 | 忍者式墙上借力 |

## 灵感来源

- **Dead Cells** — 像素美术 + 法线贴图 + 动态光照，2D 也能有顶级视觉
- **Katana Zero** — 一击必杀的手感，时间减速机制
- **Mark of the Ninja** — 忍者/武侠的 2D 动作标杆
- **Guacamelee** — 墨西哥摔跤武术，连击/投技/空中连招

### 视觉技术栈

| 技术 | 说明 | Godot 节点 |
|------|------|-----------|
| 骨骼动画 | 武术动作流畅，比逐帧更适合连招 | Skeleton2D / AnimatedSprite2D |
| 法线贴图 | 像素精灵 + 法线贴图 = 立体光照 | Sprite2D + normal_map |
| 动态光照 | 环境光 + 点光源 + 自发光 | PointLight2D / WorldEnvironment |
| 后处理 | Glow（剑气发光）+ Color Grading | CanvasModulate / ColorRect shader |
| 粒子系统 | 命中火花、剑气拖尾、受击碎片 | GPUParticles2D |
| 屏幕震动 | 打击感最便宜的手段 | Camera2D + Tween |

## 设计原则

1. **玩法第一** — 手感调好了再加美术，不要反过来
2. **纯代码先行** — 原型不需要任何外部素材就能跑
3. **边做边学** — 每个脚本注释里标注对应的学习文档
4. **文档双关联** — 文档 ↔ Blitz 源码双向引用
5. **可测试** — 任何时候 F5 都能跑，不攒功能
6. **打击感 > 复杂度** — 屏幕震动 + 击停帧 + 粒子 > 100 种招式

## 打击感四要素

参考 Dead Cells / Katana Zero 的做法：

| 要素 | 做法 | 成本 |
|------|------|------|
| 屏幕震动 | Camera2D offset Tween，重击震 4px，轻击震 1px | 0 素材 |
| 击停帧 (Hit Stop) | 被击中时双方暂停 0.05-0.1s | 0 素材 |
| 粒子特效 | 命中火花、剑气拖尾、受击碎片 | 0 素材（程序化） |
| 音效 | 挥剑声、命中声、格挡声、死亡声 | 需要素材 |

先把前三样做好，音效后面再加。**屏幕震动是所有打击感中最便宜、最有效的。**
