# Blitz — 2D Run & Gun Platformer

> Working Title — 最终名称待定

## 一句话

2D 横版跳跃躲避射击游戏。操作一个角色在平台间穿梭，射击敌人，格挡弹幕，击败 Boss。

## 灵感来源

- **茶杯头 (Cuphead)** — Run & Gun 核心玩法、Boss 战、格挡 (Parry) 机制
- **合金弹头 (Metal Slug)** — 横版射击的快节奏感
- **蔚蓝 (Celeste)** — 精确的平台跳跃手感和可访问性设计

## 核心机制

| 操作 | 按键 | 说明 |
|------|------|------|
| 移动 | A/D 或 ←/→ | 水平移动,有加速度和惯性 |
| 跳跃 | Space/W/↑ | 可变高度（轻按=小跳，长按=大跳） |
| 冲刺 | Shift/K | 短暂无敌帧的横向爆发 |
| 射击 | J/Z/鼠标左键 | 按住连发，子弹直线飞行 |
| 格挡 | 空中靠近粉色物体按跳 | 高弹跳 + 无敌帧 + 加分 |

## 美术路线图

```
v0.1 (当前)     主角=蓝色圆点   敌人=彩色圆点   平台=矩形   纯代码绘制
v0.3            主角=简单几何人 敌人=色块角色   加入粒子特效
v0.5            主角=像素小人   敌人=像素角色   加入简单动画
v1.0            主角=手绘角色   敌人=手绘角色   完整动画 + 视差背景
```

**原则**: 玩法先行，美术逐步迭代。任何时候游戏都是可玩的。

## 技术栈

- **引擎**: Godot 4.6 (Forward+ 渲染器)
- **语言**: GDScript (强类型)
- **物理**: CharacterBody2D + Area2D (无 RigidBody)
- **UI**: Godot Control 节点 (CanvasLayer)
- **碰撞**: 6 bit 分层系统 (见 main.gd 顶部注释)
- **艺术**: 全部 `_draw()` 程序化绘制 (v0.1)

## 设计原则

1. **玩法第一** — 手感调好了再加美术,不要反过来
2. **纯代码先行** — 不需要任何外部素材就能跑
3. **边做边学** — 每个脚本注释里标注了对应的学习文档
4. **可测试** — 任何时候 F5 都能跑,不攒功能
5. **简单 > 复杂** — 先做到"好玩",再加"深度"
