# Python Learning Project

Django 学习项目，用于练习 Python 脚本和 Web 开发。

## 快速启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动开发服务器
python manage.py runserver
```

## 目录结构

```
apps/python/
├── manage.py           # Django 管理脚本
├── pyproject.toml      # 项目配置
├── requirements.txt    # 依赖列表
├── project/            # Django 项目配置
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
└── apps/               # Django 应用
    └── apps.py
```
