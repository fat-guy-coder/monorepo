# 模型驱动的元编程 API 引擎

这是一个基于 Express.js 构建的高级元系统。它允许您通过调用 API 接口，动态地、分步骤地创建数据模型，并为这些模型按需生成功能完备的、符合 RESTful 风格的 CRUD API 接口。

## 功能

- **Vue.js 控制台**: **(新)** 提供一个功能完备的 Vue.js 单页面应用作为管理控制台，用于直观地创建、查看和删除模型与接口。
- **动态模型创建**: 通过 API 定义详细的 Mongoose Schema，自动生成模型文件。
- **按需接口生成**: 为已存在的模型按需生成任意组合的 CRUD 接口。
- **层级化路由**: 支持创建层级化的路由结构。
- **全动态响应消息**: API 响应消息由中央配置文件驱动。
- **配置字典** & **动态中间件配置** & **API 驱动的重启**。
- **可配置的实时日志流 (SSE)**。

## 如何启动项目

1.  **安装依赖**: `npm install`
2.  **启动服务器**: `npm start` (此命令使用 `nodemon` 监视文件变化并自动重启)

## Vue.js 管理控制台

启动服务器后，直接在浏览器中访问根路径即可打开 Vue.js 管理控制台：

[http://localhost:3000/](http://localhost:3000/)

这个控制台提供了完整的图形化界面来操作元 API 引擎：

1.  **动态表单**: 位于页面顶部的表单会根据您是“新增模型”还是“新增接口”来动态切换输入字段。
2.  **模型管理**: 
    - 点击“新增模型”按钮，填写模型名称和 Schema (JSON 格式)即可创建一个新的数据模型。
    - “已存在的模型”列表会实时展示所有 `models/` 目录下的模型，并提供“删除”按钮。
3.  **接口管理**:
    - 点击“新增接口”按钮，您可以为任何一个已存在的模型选择要生成的接口（create, getAll 等）并指定路由路径。
    - “已生成的接口”列表会实时展示所有 `routes/` 目录下的 API 文件，并提供“删除”按钮。
4.  **自动刷新**: 所有创建或删除操作成功后，列表都会自动刷新。
