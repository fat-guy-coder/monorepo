# 页面 1：游戏设备博物馆

## 布局与交互逻辑
这是一个全屏视差滚动页面的第一部分。用户向下滚动将平滑过渡到第二页。

-   **主视觉区**：占据屏幕上方约 3/4 的区域，居中显示当前选中的游戏设备。设备本身由精细的线条构成，呈动画风格。
-   **时间线区**：位于屏幕底部。一根横向的时间轴，上面密布着从 1995 到 2026 年的点状节点。

## 设备与时间线交互
1.  **初始状态**：页面加载时，默认显示**小霸王游戏机**组合。
2.  **高亮节点**：时间线上，1995、2011、2015 三个节点以特殊样式（如更大、更亮、带有光晕）高亮。鼠标悬停在这三个节点上，会触发主视图的设备切换。
3.  **设备详情**：
    -   **1995 - 小霸王**：一个老式显像管电视机，连接着两个有线手柄。电视机屏幕里播放（或静态显示）《魂斗罗》的标志性画面。整个组合使用黑白灰配色，带有细微的“雪花点”噪点效果，还原年代感。
    -   **2011 - PSP**：一台线条简约、色彩明快的 PSP 掌机。摇杆和按键清晰可见。屏幕显示《怪物猎人P3》的狩猎场景。主题色可加入活力的橙色或青色。
    -   **2015 - 笔记本电脑**：一台游戏本，键盘细节丰富。屏幕显示《英雄联盟》的团战画面或标志性英雄。
4.  **设备切换动效**：切换设备时，旧设备以快速旋转缩小的方式消失，新设备以弹性放大的方式出现，中间可有光效闪烁。
5.  **键盘交互**：设备上的每个按键（如方向键、ABXY键）都必须可点击。点击时，按键本身有“按下”的动画，同时屏幕上出现一个简单的“POW！”、“BIU！”等漫画打击特效文字。
6.  **其他时间节点**：时间线上1995-2026间的每一个点，代表该年份最具代表性的一款游戏（AI可自行搜索填充，例如：1996《精灵宝可梦红/绿》、1998《塞尔达传说：时之笛》等）。点击任意节点，页面平滑滑动到第二页，并自动筛选出该游戏的卡片。

## 代码结构要求
-   HTML 结构语义化。
-   CSS 使用 Grid 和 Flexbox 布局。
-   设备切换逻辑用 JavaScript 实现，管理设备数据和状态。
-   键盘特效可动态创建绝对定位的元素。
